问题：运行测试时出现 "Connection refused" 错误

原因：RPC 服务器未启动

解决方案：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式一】一键测试（推荐）

    ./快速测试.sh

    这会自动启动服务器并运行测试。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式二】手动启动

    终端 1 - 启动服务器：
    ./start_rpc_server.sh

    终端 2 - 运行测试：
    python test_anti_detection.py --test basic

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

首次使用请先安装浏览器：
    playwright install chromium

详细说明请查看：使用指南.md
